# Pickleball Scoreboard

A mobile-friendly pickleball scoreboard built with React 18, TypeScript, Vite, and Vitest.

## Local development

```bash
npm install
npm run dev
```

## Tests

```bash
npm test
```

The game logic in `src/core/` is covered with Vitest unit tests.

## GitHub Pages

Once the deploy workflow runs on `main`, the site will be available at:

https://anne-lin-www.github.io/Pickleball-Scoreboard/
